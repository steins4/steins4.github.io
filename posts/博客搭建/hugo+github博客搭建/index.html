hugo+github博客搭建

​ 所有的步骤都是在mac os环境中进行的。

  1. brew安装hugo

    brew install hugo

    使用brew直接安装hugo，然后通过输入hugo version命令查看hugo的版本

  2. 建立博客文件夹

    hugo new site blog

    这行命令表示会在电脑的blog文件夹中创建博客

  3. 博客主题

    1. 首先到博客主题网站去下载喜欢的博客，并将主题下载到blog的themes的文件夹下面
    2. 然后按照博客作者的设置步骤对blog文件夹的config.toml等文件进行设置

  4. 博客网站本地显示

    hugo server -t 主题名字 --buildDrafts

    在浏览器输入localhost:1313就可以在本地访问

  5. hugo连接github

    1. 首先在github上创建username.github.io的仓库，username必须和用户名一致

    2. 之后在blog目录下输入

      hugo --theme=主题名字 --baseUrl="https://username.github.io/ --buildDrafts

      会创建一个public的文件夹

    3. 进入public文件夹，依次输入

      git init
git add .
git commit -m "随便"
git remote add origin 仓库的地址
git push -u origin master

      之后便可以访问https://username.github.io查看自己的博客了
