Shell脚本

本文所说的脚本是以sh为后缀名的脚本

  1. 运行方法

    1. 第一种方法使用chmod +x ./test.sh让脚本具有执行能力，然后使用./test.sh 执行脚本
    2. 第二种方法使用/bin/bash test.sh运行脚本,不过需要在脚本开头加上#! /bin.bash

  2. Shell变量

    1. 定义变量

      1. name=“steins”其中等号前后不能有空格，
      2. 变量命名只能使用英文字母，下划线和数字，并且数字不能在首位
      3. 不能有空格，关键字，标点符号，

    2. 语句赋值

      1. for file in 'ls /etc' 或for file in $(ls /etc)

    3. 使用变量

      1. 直接在变量前加美元符号$即可
      2. echo $name或echo ${name}

    4. 只读变量

      myUrl="www.baidu.com"
readonly myUrl
      1. myUrl为只读变量，不能对其更改

    5. 删除变量

      1. unset variable_name
      2. unset不能删除只读变量

    6. 字符串

      1. 单引号和双引号
        1. 单引号的任何字符都回原样输出，单引号里面不能有变量，不能出现转义字符，双引号可以
        2. 输入：
        3. 输出：
      2. 获取字符串长度
        1. ${#variable_name}
      3. 提取子字符串
        1. ${variable_name:1:4},查找索引为1到4的字符串
      4. 查找子字符串
        variable_name="hello world"
echo `expr index "${variable_name}" io`
# 输出4

    7. Shell数组

      bash只支持一维数组，并没有限制数组的大小

      1. 定义数组
        1. array_name=(value0 value1 ... valuen)
        2. 数组中元素用空格分割
      2. 读取数组
        1. echo ${array_name[@]}
        2. 使用@可以获取数组中的所有元素
      3. 获取数组长度
        1. echo ${#array_name[0]}

  3. 传递参数

    1. echo "传递参数实例"
echo “执行的文件名：$0“
echo “执行的文件名：$1“
echo “执行的文件名：$2“
echo “执行的文件名：$3“

      ./test.sh 1 2 3

    2. 其中如果输入的是1 2 3，那么$*表示“1 2 3”， $@表示“1”，“2”，“3”

    1. 显示变量

      1. 输入：

        #！ /bin/bash
read name 
echo "$name it is a good name"

      2. 输出：

    2. 显示不换行
      1. #！ /bin/bash
echo -e "OK! \c"  # -e表示开启转义，\c表示不换行
echo "It is a test"

      2. 输出：

        OK！It is a test
